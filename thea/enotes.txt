Packages description:
---------------------

1- fr.unice.bioinfo.thea
************************

This package contains the core of thea. Everything 
we need to run Thea using the netbeans platform.

This module will always allow users to load an ontology 
and visualize it using the Ontology Explorer included 
module. (See 2).

2- fr.unice.bioinfo.thea.ontologyexplorer
*****************************************

This package allow users to visualize an ontology using 
an explorer: the ontology is presented as a "tree".

3- fr.unice.bioinfo.thea.resources
**********************************

This package contains only resources used in the package 
fr.unice.bioinfo.thea

4- fr.unice.bioinfo.thea.welcome
********************************

Utilities to show a welcome page when Thea's launched.
-------------------------------------------------------



Date: 19 Janv. 2005.
====================

Use a user-defined JDBC connection within Hibernate to
get a Session:

1- Get a java.sql.Connection instance:

	java.sql.Connection c = datasource.getConnection();

2- Get a SessionFactory instance (call it sessions)
	
	// open your session now
	Session session = sessions.openSession(c);
